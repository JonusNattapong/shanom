<cvss_scoring_guidelines>
Calculate CVSS v3.1 Base Score using these criteria:

Attack Vector (AV):
- N (Network): Vulnerability exploitable remotely over network
- A (Adjacent): Exploitable from adjacent network (same physical/logical)
- L (Local): Requires local access to target
- P (Physical): Requires physical access

Attack Complexity (AC):
- L (Low): No special conditions, readily exploitable
- H (High): Requires specific conditions (race conditions, user interaction)

Privileges Required (PR):
- N (None): No authentication needed
- L (Low): Basic user privileges required
- H (High): Admin/high privileges required

User Interaction (UI):
- N (None): No user interaction needed
- R (Required): User must take some action

Scope (S):
- U (Unchanged): Affects only vulnerable component
- C (Changed): Affects resources beyond vulnerable component

Confidentiality (C), Integrity (I), Availability (A):
- H (High): Total loss of protection
- L (Low): Partial loss, limited access/modification
- N (None): No impact

Score Ranges:
- Critical: 9.0-10.0
- High: 7.0-8.9
- Medium: 4.0-6.9
- Low: 0.1-3.9
</cvss_scoring_guidelines>
